🌐 دليل النشر الدائم

مشاركة منصة المسابقات الرياضية مع العالم - مجاناً وإلى الأبد

🎯 الهدف: نشر التطبيق بشكل دائم ومجاني على الإنترنت

بدون تغيير أو إضافة أو حذف أي شيء من التطبيق الحالي

1 🚀 إنشاء حساب Render

• اذهب إلى: https://render.com

• اضغط "Get Started for Free"

• سجل باستخدام GitHub (الأسهل والأسرع)

• مجاني 100% - 750 ساعة شهرياً

2 📁 رفع الكود إلى GitHub

• اذهب إلى: https://github.com/new

• اسم Repository: math-competition-platform

• اجعله Public (مهم للخطة المجانية)

• ارفع جميع ملفات المشروع كما هي

3 🔗 ربط Render بـ GitHub

• في Render Dashboard: اضغط "New +"

• اختر "Web Service"

• اضغط "Connect a repository"

• اختر المشروع من GitHub

4 ⚙️ إعدادات الخدمة
Name: math-competition-platform Environment: Python 3 Region: Oregon (US West) Branch: main Build Command: pip install Django==5.2.1 gunicorn==21.2.0 whitenoise==6.6.0 && python manage.py migrate --noinput Start Command: gunicorn alhassan.wsgi:application --bind 0.0.0.0:$PORT --workers 1 --timeout 120
5 🔐 متغيرات البيئة
DJANGO_SETTINGS_MODULE = alhassan.simple_settings DEBUG = False STUDENT_ACCESS_CODE = ben25 SECRET_KEY = [سيتم إنشاؤها تلقائياً] PORT = 10000 PYTHONPATH = . DISABLE_COLLECTSTATIC = 1
6 💾 قاعدة البيانات

• اضغط "New +" → "PostgreSQL"

• Name: math-competition-db

• Plan: Free

• سيتم ربطها تلقائياً

7 ⏱️ انتظار النشر

• سيظهر "Building..." ثم "Live" (5-10 دقائق)

• ستحصل على رابط مثل:

https://math-competition-platform.onrender.com
8 🔑 إنشاء مدير جديد

• في Render Dashboard → اختر الخدمة

• اضغط "Shell" في الأعلى

python manage.py createsuperuser

• أدخل البيانات المطلوبة
